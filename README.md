# MakemyTrip
- Live Link : https://makemytrip-beta.vercel.app/

## Project Overview
This project is an AI-powered vacation planning assistant.  
It helps users plan their trips by suggesting destinations, generating travel plans, and managing bookings.

The system combines a modern frontend interface with a backend API and AI integration to provide a smart travel planning experience.

 ## Technologies Used

### Frontend
- ReactJS
- HTML
- CSS
- JavaScript

### Backend
- Spring Boot
- Java

### Database
- MySQL
